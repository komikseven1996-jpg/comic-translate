# Custom Fonts

Letakkan semua varian dari satu keluarga font di folder ini.

Nama file yang disarankan:

- `NamaFont-Regular.ttf`
- `NamaFont-Italic.ttf`
- `NamaFont-Bold.ttf`
- `NamaFont-BoldItalic.ttf` (opsional)

Format yang didukung untuk disiapkan: `.ttf` atau `.otf`.

Jangan mencampur beberapa keluarga font dengan nama internal yang sama karena
Qt memilih varian berdasarkan metadata internal font, bukan hanya nama file.
